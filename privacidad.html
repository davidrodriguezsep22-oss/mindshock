Política de privacidad

Última actualización: julio de 2026.

Información recibida

Durante la fase beta, MindShock puede recibir datos que el usuario comparte voluntariamente por WhatsApp o correo, como nombre, experiencia laboral, estudios y documentos necesarios para prestar el servicio.

Uso de la información

La información se utiliza exclusivamente para preparar los documentos o servicios solicitados y comunicarse con el usuario.

Protección de datos

No se vende información personal. El usuario debe evitar enviar contraseñas, datos bancarios completos u otra información que no sea necesaria para el servicio.

Eliminación

El usuario puede solicitar la eliminación de sus archivos y datos de trabajo contactando a MindShock.
