MINDShock - Página web mejorada

Contenido:
- index.html: página completa mejorada, con estilos, formulario y conexión a Supabase integrados.

Cómo subirla a GitHub:
1. Descomprime este archivo ZIP.
2. Abre el repositorio mindshock.
3. Pulsa Add file > Upload files.
4. Arrastra el archivo index.html.
5. GitHub debe reemplazar el index.html existente.
6. Confirma Commit directly to the main branch.
7. Pulsa Commit changes.

Importante:
El index.html actual del repositorio solo redirige a chatgpt.site.
Para que mindshock.app muestre directamente la página mejorada, sí es necesario reemplazar ese index.html.
