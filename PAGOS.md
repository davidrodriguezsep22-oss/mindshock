# MindShock: estructura de pagos

- `index.html`: sitio público, sin credenciales ni código Sandbox.
- `pago.html`: página privada para enlaces de pago PayPal Live.
- `pruebas/paypal-sandbox.html`: página aislada para pruebas internas.
- `robots.txt`: evita que las páginas de pago y prueba se indexen.

Los secretos nunca se guardan en GitHub.
